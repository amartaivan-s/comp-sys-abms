Pedestrian-Car Crossing Simulation

This demo illustrates Agent-Based Modeling (ABM) applied to a complex urban crossing system. Autonomous pedestrians move towards their target points while avoiding collisions with cars. Lanes are controlled by traffic lights, and agents obey traffic rules dynamically.

  • Pedestrians start at designated points (top/bottom) and move to their targets.
  • Cars move along lanes with traffic lights controlling flow.
  • Collision avoidance occurs between pedestrians and vehicles.
  • The environment contains lanes, start/target points, and traffic lights.

About Mesa

This model was implemented with Mesa, a Python framework for Agent-Based Modeling. Mesa allows building, visualizing, and analyzing ABMs in Python, capturing complex system behaviors through local agent interactions.

About Solara

The interactive simulation interface is built using Solara, a Python framework for building web applications and dashboards. Solara allows you to create reactive, interactive visualizations with Python without writing JavaScript, integrating seamlessly with simulations, charts, and HTML/SVG elements.

In this demo, Solara is used to render the pedestrian-car crossing environment, including agents, lanes, traffic lights, and control widgets such as sliders and buttons.
